Forex M

Live foreign exchange monitor

A live foreign exchange dashboard pulling real rates from the Frankfurter API (European Central Bank data) for 39 currencies across all major global regions — with currency strength analysis, candlestick charts, and a live converter.

  • Live ECB rates via Frankfurter API, graceful fallback to simulated data
  • 39 currencies across Americas, Europe, Asia, Africa, Oceania, Middle East
  • Currency strength index ranking 8 majors (USD, EUR, GBP, JPY, CHF, CAD, AUD, NZD)
  • Candlestick OHLC charts per pair via Lightweight Charts library
  • Base currency switcher with 12 quick-select options and custom amount input
  • Pip calculator, bid/ask spread, 24h high/low, inline sparklines
  • Forex session tracker: Sydney, Tokyo, London, New York — overlap detection
  • Region filter, sort by rate/change/pips, favorites with localStorage
